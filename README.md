# AMD-Module-Demo
- no-require-demo ： 最原始html 引入js的demo
- compatible-demo : 非AMD 模块标准，兼容导出
- require-demo ： 定义AMD模块，使用require导入demo 
- require-config-demo: 定义AMD模块，配置require-config引入多模块，最后使用require导入demo
